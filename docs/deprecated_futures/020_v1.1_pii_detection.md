# PII Detection & Redaction

> **Versão Target:** V1.1

## Descrição

Clientes podem compartilhar dados sensíveis (CPF, cartão de crédito) nas mensagens. Compliance com LGPD/GDPR requer proteção.

---

## Passos de Implementação

### 1. Código

- [ ] `src/utils/pii_detector.py`: Detecção de PII usando regex + spaCy NER
  - CPF, RG, cartão de crédito, email, telefone, endereço
- [ ] `src/utils/sanitization.py`: Integrar PII redaction
- [ ] `src/api/ingest_routes.py`: Redact PII antes de storage
- [ ] `src/models/interaction.py`: Campo `pii_detected` boolean

### 2. Testes

- [ ] Unit tests: PII detection accuracy
