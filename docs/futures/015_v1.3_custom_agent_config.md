# Custom Agent Configuration Per Company

> **Versão Target:** V1.3
> **Status:** ⏳ Pendente
> **Owner:** Unassigned
> **Estimativa:** 40h

---

## Descrição

Todas empresas usam o mesmo pipeline de 4 agentes com a mesma lógica. Esta feature permite customização de agentes por empresa.
